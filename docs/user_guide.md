# 问年 WenNian 用户指南 v2.0.0

## 简介
问年是基于多智能体协作的十二维度衰老元模型与工业决策引擎。

## 快速开始
1. 访问 Gradio 界面: http://localhost:7860
2. 输入年龄和血检指标
3. 点击「开始评估」
4. 查看报告并下载PDF

## 所需指标
基本评估需要9项指标：
- 年龄、白蛋白、肌酐、空腹血糖
- 淋巴细胞百分比、MCV、RDW
- 碱性磷酸酶、白细胞计数

可选指标：
- C反应蛋白、收缩压、舒张压

## 报告解读
- **生物年龄**: 基于4个时钟融合的评估值
- **衰老加速**: 生物年龄 − 日历年龄
- **主驱动维度**: 最需关注的衰老方向
